# Hydro Pacifique

## Requirements

- MySQL
- Apache + PHP

## Run the application

From the project root:

`php -S 127.0.0.1:8080 -t web_app`

Then open:

[http://127.0.0.1:8080/login.php](http://127.0.0.1:8080/login.php)

## Run with Docker

From the project root:

`docker compose up --build`

Then open:

[http://127.0.0.1:8080/login.php](http://127.0.0.1:8080/login.php)
